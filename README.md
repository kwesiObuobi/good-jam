# ğŸ“– Good Jam <a name="about-project"></a>

**Good Jam** is a project built as part of an exercise at Microverse Inc. As a capstone project, this is a fictional annual event that happens in May of every year, where top musicians hold a conference to discuss what makes a quality music what it is. Microverse's idea of this project is give its students a taste of how real-world projects look like.
